Quick Setup Guide

Get started in just 5 minutes with these simple steps

1

Copy Workflow File

Create .github/workflows/update-stats.yml in your profile repository and copy our workflow configuration.

mkdir -p .github/workflows
curl -o .github/workflows/update-stats.yml https://raw.githubusercontent.com/alessandrobrunoh/update-profile-stats-script/main/example-workflow-for-users.yml
2

Add API Keys

Go to your repository Settings ‚Üí Secrets and add your GitHub token and Gemini API key.

GITHUB_TOKEN Personal Access Token with repo permissions
GEMINI_API_KEY Google AI Studio API key for analysis
3

Run & Enjoy

Commit the workflow file and watch as your profile gets automatically updated with beautiful statistics!
